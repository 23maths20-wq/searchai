# 🔍 SearchAI

An AI-powered search engine with multiple LLM backends (Groq & OpenRouter).

## ✨ Features

- 🤖 AI-powered instant answers
- ⚡ Powered by Groq (Llama 3.3 70B) - Ultra fast responses
- 🔄 Multi-API backup with OpenRouter
- 📱 PWA - Install as an app
- 🌙 Dark mode support
- 💬 Chat interface
- 📺 YouTube search integration
- 🔒 Firebase authentication
